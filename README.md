# Corridor Key Pro

An FxPlug4 plug-in that brings [CorridorKey][corridor-key]'s AI green-screen keying to Final Cut Pro on Apple Silicon.

[corridor-key]: https://github.com/nikopueringer/CorridorKey

---

## Repository layout

| Path             | What lives there                                                        |
| ---------------- | ----------------------------------------------------------------------- |
| `Source/`        | The Swift / Metal source tree and Xcode project for Corridor Key Pro.   |
